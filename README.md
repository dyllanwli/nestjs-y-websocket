# Integrating Yjs on NestJS 
with y-websocket



## y-client 
simple y-quill collaborative editor client
```
cd y-client
npm i
npm run start
```
then open http://localhost:8080/quill.html

## start demo
```
npm i 
npm run start:dev
```
